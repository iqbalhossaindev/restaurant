Dish Studio Website

Files:
index.html
styles.css
script.js
assets/

How to use:
1. Unzip the project.
2. Open index.html in your browser.
3. To use your own dish cutout images, replace the files inside assets/ and keep the same file names, or update the image paths in script.js.
4. For the cleanest look, use transparent PNG or SVG dish images.

Animation in this version:
1. Current dish exits to the left.
2. Next dish enters from the top.
3. Motion is smooth with a rounded path feel.
